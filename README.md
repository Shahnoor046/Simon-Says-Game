# Simon-Says-Game

A simple and interactive Simon Says Game built using HTML, CSS, and JavaScript.

## About the Project

Simon Says is a memory-based game where the player has to remember and repeat the sequence generated by the game. With each level, the sequence becomes longer and more challenging.

## Features

* Random color sequence generation
* Level progression system
* User input validation
* Game over detection
* Interactive UI effects
* Built using JavaScript DOM Manipulation

## Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## How to Play

1. Press any key to start the game.
2. Watch the highlighted button carefully.
3. Repeat the sequence by clicking the buttons.
4. Each level adds a new step to the sequence.
5. The game ends when you click the wrong button.
